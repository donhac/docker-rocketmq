# RocketMQ Sample
rocketmq-docker-sample for 4.2.0 version

+ rocketmq-namesrv: RocketMQ Name Server
+ rocketmq-broker-a-m: RocketMQ Broker
+ rocketmq-console: UI for visualization Pub & Sub

### Usage:

#### RocketMQ Console
```
$ cd rocketmq-console
$ mvn clean package
```

#### docker-compose:
```
docker-compose up -d
```

#### RocketMQ Console
http://localhost:8080
